☁️ Cloud Sync

OFF

Đồng bộ dữ liệu MeYeuBe qua Supabase. V10.1 dùng cơ chế đồng bộ nguyên JSON để nhiều thiết bị dùng chung dữ liệu nhanh, vẫn giữ localStorage để chạy offline.

Chưa bật đồng bộ Nhập Supabase URL, Publishable key và Sync ID rồi bấm Lưu cấu hình.
ONLINE

Lưu ý: Thiết bị có dữ liệu đầy đủ nhất nên bấm Đẩy lên Cloud trước. Thiết bị khác nhập cùng Sync ID rồi bấm Tải Cloud về. V10.1 dùng 1 bản JSON chung, nên hạn chế nhập đồng thời trên nhiều thiết bị cho đến khi nâng cấp đồng bộ theo bảng.

Dữ liệu & sao lưu

Dữ liệu lưu trong trình duyệt Safari trên iPhone. Nên xuất DB định kỳ, đặc biệt trước khi xoá lịch sử web hoặc đổi máy.

Cảnh báo: thao tác này sẽ xoá toàn bộ dữ liệu đang lưu trên trình duyệt hiện tại. Boss nên xuất DB JSON trước khi xoá.
